```python
class ProcessingError(Exception):
    """Error durante procesamiento del pipeline. Incluye step donde fallo."""
    def __init__(self, step: str, detail: str):
        self.step = step
        self.detail = detail
        super().__init__(f"[{step}] {detail}")
```

**Step 1 - decode_and_validate(image_bytes: bytes, config: AppConfig) -> tuple[Image.Image, str, str]:**
- `Image.open(BytesIO(image_bytes))` dentro de try/except (PIL.UnidentifiedImageError, Exception) -> raise ProcessingError("decode", "Invalid or corrupt image") per D-02
- Verificar megapixels: `w * h > 25_000_000` -> raise ProcessingError("decode", f"Image {w}x{h} exceeds maximum 25 megapixels") per D-05
- `ImageOps.exif_transpose(img)` — SIEMPRE, antes de todo (per PIPE-02, D-08)
- Si img.mode == "CMYK": `img = img.convert("RGB")` (per D-07)
- `img = img.convert("RGBA")` — necesario para el resto del pipeline
- Loggear JSON structured: `{"level": "info", "event": "decode_complete", "article_id": article_id, "original_size": f"{w}x{h}", "mode": original_mode}` (per D-03, D-04)
- Retorna (img_rgba, f"{original_w}x{original_h}", original_mode)

**Step 2 - remove_background(img: Image.Image, config: AppConfig, rembg_session) -> Image.Image:**
- Deteccion de alpha pre-existente (per D-06): contar pixeles con alpha < 128 en el canal alpha. Si `transparent_pixels / total_pixels > 0.10` -> skip rembg, retornar img directamente
- Convertir img a bytes PNG en BytesIO
- `result_bytes = rembg.remove(input_bytes, session=rembg_session, alpha_matting=config.rembg.alpha_matting, alpha_matting_foreground_threshold=config.rembg.alpha_matting_foreground_threshold, alpha_matting_background_threshold=config.rembg.alpha_matting_background_threshold, alpha_matting_erode_size=config.rembg.alpha_matting_erode_size)`
- `result = Image.open(BytesIO(result_bytes)).convert("RGBA")`
- Loggear JSON: `{"level": "info", "event": "rembg_complete", "duration_ms": ..., "skipped": false}`
- Retorna result

**Step 3 - autocrop(img: Image.Image, config: AppConfig) -> Image.Image:**
- Si `config.autocrop.enabled == False` -> retornar img sin cambios
- `alpha = img.split()[3]`
- Binarizar: `alpha = alpha.point(lambda p: 255 if p > config.autocrop.threshold else 0)`
- `bbox = alpha.getbbox()`
- Si bbox is None -> retornar img sin cambios
- Guard: si area del bbox < 5% del area total (`(bbox[2]-bbox[0])*(bbox[3]-bbox[1]) / (img.width*img.height) < 0.05`) -> retornar img sin cambios, loggear warning
- `img = img.crop(bbox)`
- Loggear JSON: `{"level": "info", "event": "autocrop_complete", "bbox": bbox, "cropped_size": f"{img.width}x{img.height}"}`
- Retorna img

**Step 4 - calculate_scale_and_position(img: Image.Image, config: AppConfig) -> tuple[int, int, int, int]:**
- `canvas_px = config.output.size` (800)
- `padding_px = int(canvas_px * config.padding.percent / 100) if config.padding.enabled else 0`
- `available_px = canvas_px - (2 * padding_px)` (640 con padding 10%)
- `scale = min(available_px / img.width, available_px / img.height)`
- `scaled_w = int(img.width * scale)`
- `scaled_h = int(img.height * scale)`
- `offset_x = (canvas_px - scaled_w) // 2`
- `offset_y = (canvas_px - scaled_h) // 2`
- Retorna (scaled_w, scaled_h, offset_x, offset_y)

Agregar a `app/processor.py` las funciones restantes:

**Step 5 - composite(img: Image.Image, config: AppConfig) -> Image.Image:**
- Recibir img (RGBA con producto cropped)
- Calcular escala y posicion llamando a `calculate_scale_and_position(img, config)`
- `resample = Image.LANCZOS if scale <= 1.0 else Image.BICUBIC` (LANCZOS para downscale, BICUBIC para upscale). Calcular `scale = min(available_px/img.width, available_px/img.height)` localmente para decidir resample
- `scaled = img.resize((scaled_w, scaled_h), resample)`
- `canvas = Image.new("RGB", (config.output.size, config.output.size), tuple(config.output.background_color))`
- `canvas.paste(scaled, (offset_x, offset_y), mask=scaled.split()[3])` — usar canal alpha como mascara
- Loggear JSON: `{"level": "info", "event": "composite_complete", "canvas_size": f"{config.output.size}x{config.output.size}", "product_size": f"{scaled_w}x{scaled_h}"}`
- Retorna canvas (RGB, sin alpha)

**Step 6 - enhance(img: Image.Image, config: AppConfig) -> Image.Image:**
- Si `config.enhancement.brightness == 1.0 and config.enhancement.contrast == 1.0` -> retornar img sin cambios (optimizacion)
- Si brightness != 1.0: `img = ImageEnhance.Brightness(img).enhance(config.enhancement.brightness)`
- Si contrast != 1.0: `img = ImageEnhance.Contrast(img).enhance(config.enhancement.contrast)`
- Loggear JSON: `{"level": "info", "event": "enhance_complete", "brightness": config.enhancement.brightness, "contrast": config.enhancement.contrast}`
- Retorna img

**Step 7 - encode_webp(img: Image.Image, config: AppConfig) -> bytes:**
- `buf = BytesIO()`
- Si `config.output.quality == 0`: `img.save(buf, format="WEBP", lossless=True)`
- Else: `img.save(buf, format="WEBP", quality=config.output.quality)`
- Loggear JSON: `{"level": "info", "event": "encode_complete", "format": "webp", "size_bytes": buf.tell()}`
- `return buf.getvalue()`

**Funcion principal - process_image(image_bytes, article_id, config, rembg_session) -> ProcessingResult:**
- `import time; start = time.monotonic()`
- `steps_applied = []`
- Step 1: `img, original_size, original_mode = decode_and_validate(image_bytes, config)` -> `steps_applied.append("decode")`
- Step 2: `img = remove_background(img, config, rembg_session)` -> `steps_applied.append("rembg")`
- Step 3: `img = autocrop(img, config)` -> `steps_applied.append("autocrop") if config.autocrop.enabled`
- Step 4+5: `img = composite(img, config)` -> `steps_applied.append("scale"); steps_applied.append("composite")`
- Step 6: `img = enhance(img, config)` -> `steps_applied.append("enhance") if brightness!=1.0 or contrast!=1.0`
- Step 7: `result_bytes = encode_webp(img, config)` -> `steps_applied.append("encode")`
- `elapsed_ms = int((time.monotonic() - start) * 1000)`
- Retornar `ProcessingResult(image_bytes=result_bytes, article_id=article_id, processing_time_ms=elapsed_ms, model_used=config.rembg.model, original_size=original_size, output_size=f"{config.output.size}x{config.output.size}", steps_applied=steps_applied)`
- Wrappear TODO en try/except: si ProcessingError -> re-raise; si cualquier otra Exception -> raise ProcessingError(step="unknown", detail=str(e)) per D-09

**IMPORTANTE:** process_image es una funcion SINCRONA (no async). La llamada async se hace desde queue.py con asyncio.to_thread(). La firma del PRD dice async pero el procesamiento real es CPU-bound sincronico.
