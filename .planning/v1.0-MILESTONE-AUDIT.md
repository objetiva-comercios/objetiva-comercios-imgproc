## Tech Debt Summary

### TECH-DEBT-01 — scipy/numpy undeclared (MEDIUM)

**File:** `requirements.txt`
**Impact:** `app/processor.py:183-184` imports `scipy.ndimage` and `numpy` for `_clean_alpha_artifacts()`. Currently available as transitive deps of `rembg[cpu]` via `scikit-image`. If rembg changes its dep tree, processor fails with `ModuleNotFoundError`.
**Fix:** Add `scipy>=1.16` and `numpy>=2.3` to `requirements.txt`.
**Affects:** PIPE-03, PIPE-04

### TECH-DEBT-02 — UI CDN dependencies (LOW)

**File:** `app/templates/ui.html` lines 8-9
**Impact:** Google Fonts (Inter) and unpkg.com (Lucide icons) loaded at page start. UI degrades visually in air-gapped environments. Core functionality unaffected (fallback to system-ui, lucide guard).
**Fix:** Inline font or use system stack; bundle lucide.min.js.
**Affects:** UI-01

### TECH-DEBT-03 — Model documentation mismatch (LOW)

**File:** `CLAUDE.md` constraints section
**Impact:** Docs say `birefnet-lite`, implementation uses `isnet-general-use`. Change was deliberate (birefnet-lite exceeded 2GB RAM) and documented in 01-05-SUMMARY. No runtime issue — isnet-general-use fits in 1.5GB.
**Fix:** Update CLAUDE.md constraints to reflect `isnet-general-use`.

### TECH-DEBT-04 — Pillow deprecation warning (INFO)

**File:** `app/processor.py:115`
**Impact:** `Image.Image.getdata` deprecated in Pillow 14 (expected 2027). No impact on current version (12.1.1).
**Fix:** Migrate to new API before Pillow 14.

### TECH-DEBT-05 — Human verifications pending (INFO)

3 items requiring manual verification in production:
1. Docker build completa y modelo embebido (Phase 1)
2. RAM < 2GB durante procesamiento real (Phase 1)
3. Hot-reload en container Docker con bind mount (Phase 2)
